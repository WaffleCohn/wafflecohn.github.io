Falling Boulders Game

About the Game

Description: In Falling Boulders, the object of the game is to dodge as many many boulders fall from the top of the screen. The player can only move left and right and the boulders fall pretty fast so you will have to be quick.

Instructions: Move your player (1) by pressing the arrow keys on your keyboard the Left (2) and Right (3) buttons under the game field (depending on which you choose in the preferences [the arrow keys are set to default]). For each boulder (5) you successfully avoid you earn 1 point (6). But if any single boulder hits the player the game ends. To begin the game press the Start button (4).
