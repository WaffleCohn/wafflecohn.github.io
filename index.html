PROJECTS

Falling Boulders

Falling Boulders is a game I made in middle school while learning web design. The player moves left and right to dodge a continuous wave of falling boulders.

Map Quizzes

Using a site I created to generate the code, I was able to replicate the map quizzes given in AP Human Geography to create an effective studying tool. Click to see an example map.

Flappy Bird

Following the popularity of the mobile game Flappy Bird, I made my own online version using HTML tables and JavaScript in ninth grade.

Grogger

Grogger is an HTML5 game that parodies the classic arcade game Frogger based on the Jewish holiday Purim.

Word Brain Solver

After playing the mobile game Word Brain, I decided to create a website that could solve any of the puzzles from the game.

Random Maze Generator

This website turns your screen into a randomly generated maze using a modified depth-first search algorithm. Press 'space' to toggle a highlighted solution or 'enter' to watch an animated solver.
